# codevim

Un plugin de Nvim para autocompletar y generar código utilizando Ollama y Codeqwen.

**Características:**

* Autocompleta código de forma inteligente
* Genera código nuevo
* Soporta Ollama y Codeqwen
* Fácil de instalar y configurar
